What Is a CRDT?

CRDT stands for Conflict-Free Replicated Data Type. It’s a data structure that allows multiple users to concurrently edit shared data without ever conflicting.

Key Benefits

  • Automatic merge of changes
  • No central server required
  • Offline edits seamlessly sync

Example code snippet:

const crdt = new GCounter();
// Increment locally
crdt.increment('user1');
// Merge with remote
crdt.merge(remoteCrdt);
// Read value
console.log(crdt.value());
“With CRDTs, you get strong eventual consistency without conflict resolution headaches.”
